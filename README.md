📸 Image Gallery Project
📌 Overview

This project is a simple Image Gallery Website created using basic HTML.
It displays a collection of popular cartoon images arranged in a gallery format.

🎯 Features

🖼️ Displays multiple cartoon images

📱 Responsive layout using viewport meta tag

🎨 Simple and clean design

📌 Images aligned using CSS float and spacing

🛠️ Technologies Used

HTML5

Inline CSS

📂 Project Structure
Image-Gallery/
│
├── index.html   # Main HTML file
└── README.md    # Project documentation
🚀 How to Run the Project

Download or clone the repository

Open the index.html file in any web browser

View the image gallery

⚠️ Limitations

Uses inline CSS (not scalable for large projects)

No responsiveness for different screen sizes beyond basic support

No interactivity (purely static)
